## 3. Cleanup problem

V1-07 was functionally valid, but the workspace file still carried cleanup risk:

- dead legacy state could obscure the V1 source of truth;
- the test suite still had one silent tap with `warnIfMissed: false`;
- V1-08 would be harder to build if obvious dead code remained.

A focused audit showed that `_LegacyStorylinesWorkspaceState` and `_StorylineContentTab.chapters` were already absent in the current source before code changes in this bis. The remaining cleanup was therefore intentionally narrow.

## 4. Dead code removed

Removed from `storylines_workspace.dart`:

- `import 'dart:math' as math;` after the old legacy geometry code was gone;
- `_selectedGlobalStoryId` field from `_StorylinesWorkspaceState`;
- `didUpdateWidget` logic that only maintained that removed legacy global-story selection state.
